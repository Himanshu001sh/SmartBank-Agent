# 💰 SmartBank-Agent

An AI-powered personal financial assistant built for the SBI Agentic AI Hackathon.

SmartBank-Agent helps users understand their financial health, plan goals, receive personalized recommendations, and interact with an intelligent AI financial coach powered by Gemini.

---

## 🚀 Features

### 📊 Financial Health Dashboard
- Income and expense tracking
- Financial Health Score
- Savings analysis
- Interactive visualizations

### 🎯 Goal Planner
- Set financial goals
- Goal feasibility analysis
- Monthly savings estimation
- Progress tracking

### 🤖 AI Financial Coach
- Powered by Google Gemini
- Personalized financial guidance
- Budgeting suggestions
- Savings recommendations
- Financial literacy support

### 🧠 Personalized Recommendations
- Spending insights
- Savings opportunities
- Goal-based suggestions
- Risk-aware recommendations

### 👤 Customer Profile
- User financial profile management

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Himanshu001sh/SmartBank-Agent.git
cd SmartBank-Agent
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run Application

```bash
streamlit run app.py
```
